# CodSoft Web Development Tasks

This repository contains tasks completed during the CodSoft Web Development Internship.

## Getting Started

To launch any task locally:
1. Navigate to the task directory (e.g., `cd "1st task"`).
2. Start a local HTTP server. For example:
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your web browser.
