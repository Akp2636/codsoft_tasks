# CodSoft Web Development Tasks

This repository contains tasks completed during the CodSoft Web Development Internship.

## Project Structure

*   **[1st task/](./1st%20task)** - **Personal Portfolio Website**
    *   A premium, high-impact, and fully responsive developer portfolio featuring dark slate aesthetics, floating glassmorphism UI containers, linear gradients, dynamic typing hero text, scroll-reveal behaviors, project-details modal overlays, a previewable and download-linked ATS-optimized CV, and a contact form with active submitting verification panels. Built strictly using vanilla HTML5, CSS3, and JavaScript.

## Getting Started

To launch any task locally:
1. Navigate to the task directory (e.g., `cd "1st task"`).
2. Start a local HTTP server. For example:
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your web browser.
